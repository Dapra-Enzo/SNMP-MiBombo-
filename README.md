# SAE501/2 - Projet Logiciel 

*Introduction : La SAE501/2 consiste a réalisé en binomes un outils logiciel de gestion de trames SNMP avec deux version en premier lieu SNMP v2c puis en second  Réalisation d’un outil logiciel de gestion de trames SNMP v2c (puis v3)* 


# Contexte et enjeux

Les clients expriment le besoin d’un logiciel permettant de gérer différentes requêtes SNMP (**GET, SET, TRAP, …**) sur un réseau local.  

Le projet intègre plusieurs dimensions :  
- **Technique** : développement réseau, sécurité, analyse de trames, base de données, API REST et interface graphique.  
- **Organisationnelle** : travail en binôme avec une répartition claire des rôles, gestion de projet et suivi régulier avec le client.  
- **Professionnelle** : tenue de réunions toutes les deux semaines, rédaction de comptes rendus validés, documentation utilisateur et développeur, mise à disposition d’un dépôt Git et d’une solution de déploiement.  

Ce projet a donc pour but de confronter les étudiants à une **situation réaliste de développement logiciel en équipe**, avec des contraintes techniques et organisationnelles proches du monde professionnel.
